README

Calculator 1.0.0 release
-------------------------

Modo de uso (linea de comandos):

> java -jar calculator.jar operando1 operando2
operando1 y operando2 son los numeros positivos que se desea multiplicar

Requerimientos:
---------------

SO: 			Windows, GNU/Linux
Runtime: 		JRE 6 o superior

Contenido:
----------

README.txt				Este archivo
calculator.jar			Binario ejecutable con la aplicación
/src					Codigo fuente del proyecto
/test					Pruebas unitarias del proyecto
